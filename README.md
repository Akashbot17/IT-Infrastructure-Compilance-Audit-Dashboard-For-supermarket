# VeriLedger — Supermarket IT Audit Dashboard

A professional, enterprise-grade IT Audit Dashboard built for supermarket financial and transaction auditing.

## 🔍 Features
- KPI Cards: Total Sales, Transactions, High Risk, Duplicates, Pending Approvals
- Charts: Sales by Category (Bar), Sales Trend (Line), Payment Distribution (Doughnut), Risk Heatmap
- Anomaly Detection: Flags high-value, duplicate, and off-hours transactions
- Transaction Ledger: Full table with risk scores and status (Approved / Flagged / Review)
- Risk Management: High / Medium / Low scoring logic with visual distribution
- Audit Reports: Critical, Major, Minor findings with recommendations
- Settings: Toggle alerts, notifications, retention, and access control

## 🛠 Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Chart.js 4.4.1

## 📊 Audit Scope
- Supermarket: FreshMart Chain
- Period: Q1 2026
- Findings: 3 Critical · 3 Major · 2 Minor
